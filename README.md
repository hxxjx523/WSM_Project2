# 너의 얼굴이 보고 싶어
HTML + CSS + JS를 활용한 여성향 미연시 웹 게임
</br>
</br>

## 프로젝트 소개
WSM 프로젝트에서 주제 '사람, 물건, 음식, 10대 문화 중 하나를 이동시키는 것(장소, 시간, 마음, ...)' 에서 마음 이동을 선택
</br>
</br>

## 프로젝트 특징
1. 선택지에 따른 엔딩
2. 게임 진행으로 남주의 이미지가 변경되는 스토리
3. 웹 페이지 형식의 게임
</br>
</br>
